Shortcuts

General

  • CTRL-z => History backward
  • CTRL-SHIFT-z => History forward

Hexadecimal view

  • CTRL-c => Copy the selection in the escaped hexadecimal format: "\41\42"
  • CTRL-SHIFT-c => Copy the selection in the classic hexadecimal format: "4142"
  • ALT-c => Copy the selection as a C-style array: "{ 0x41, 0x42 }"
  • CTRL-a => Select all
  • DEL => Delete selection
