Windows Binary

A Windows MSI is available HERE

The bundled Python 2.7/3.x plugins shipped with this installer need Python 2.7/3.3 (x86 version) to be installed. See http://www.python.org/getit/

If you don't want to install the Python interpreters, and want to suppress the error messages, delete the plugins files in %application dir%\plugins\.

This binary also comes with a version of OpenSSL compiled with VC2010

The VC2010 redistributables are not shipped with the installer anymore, if the application is complaining about a missing msvc* library, install the MS package from HERE
