Windows Binary

A set of Windows binaries packages are available HERE

The bundled Python 2.7/3.x plugins need Python 2.7/3.4 (x86 or x64 version) to be installed. See https://www.python.org/downloads/

If you don't want to install the Python interpreters, and want to suppress the error messages, delete the plugins files in %application dir%\plugins\.

This binary also comes with a version of OpenSSL and Distorm compiled with VC2013

If the application is complaining about a missing msvc* library, install the MS package vcredist.exe bundled in the binary package.
