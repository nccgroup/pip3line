Windows Binary

A set of Windows binaries packages are available HERE

The bundled Python 2.7/3.x plugins need Python 2.7/3.3 (x86 or x64 version) to be installed. See https://www.python.org/downloads/

If you don't want to install the Python interpreters, and want to suppress the error messages, delete the plugins files in %application dir%\plugins\.

This binary also comes with a version of OpenSSL and Distorm compiled with VC2010/VC2012

The VC2010/VC2012 redistributables are not bundled anymore, if the application is complaining about a missing msvc* library, install the MS package corresponding to your version:

  • VC2010 (x86)
  • VC2012 (x64)
  • VC2013 (x86 & x64)
